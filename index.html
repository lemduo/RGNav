PAPlanner

Progress-Aware Planner for Embodiment-Agnostic Vision-and-Language Navigation

📄 Paper PDF 💻 Code

Abstract

Vision-and-Language Navigation in Continuous Environments (VLN-CE) has achieved remarkable progress with the integration of Vision-Language Models (VLMs). However, VLM-based navigation presents a key challenge when following long-horizon instructions, where reliance on implicit temporal reasoning often leads to unreliable progress estimation. To address this limitation, we propose PAPlanner, a progress-aware planning framework for VLN-CE. We first formulate navigation as a progress-aware planning problem where each instruction is decomposed into ordered sub-instructions. Then, the VLM explicitly tracks their execution progress while predicting navigation commands within a structured action space, significantly enhancing reasoning stability. For safe deployment, these commands are finally executed by a locomotion policy that combines real-time traversability estimation with classical local planning. By decoupling semantic reasoning from motion execution, PAPlanner achieves embodiment-agnostic deployment and enables safe navigation without retraining. Extensive experiments on R2R-CE and RxR-CE benchmarks, as well as real-world deployments on humanoid, quadruped, and wheeled robots, demonstrate that PAPlanner achieves superior generalization in unseen environments and robust sim-to-real transfer.
